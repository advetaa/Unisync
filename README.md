# UniSync

UniSync is a university collaboration and management platform developed as a Project Based Learning (PBL) project.

The platform helps students manage tasks, track study sessions, organize expenses, and improve collaboration within academic teams.

---

## 🚀 Features

- User Authentication System
- Task Management
- Study Session Tracker
- Expense Tracker
- Profile Management
- Responsive Dashboard
- Database Integration
- Clean User Interface

---
